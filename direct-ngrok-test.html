⚠️ Problem: Ngrok visar en varningssida som blockerar CSS/JS

Steg 1: Öppna Staff App direkt

Klicka på länkarna nedan för att öppna i nytt fönster (kringgår ngrok-varningen):

👷 Staff Login 📊 Staff Dashboard

Steg 2: Om det fortfarande inte fungerar

Prova att lägga till detta i URL:en: ?ngrok-skip-browser-warning=true

🔧 Staff med skip parameter

Steg 3: Testa kameran

  1. Logga in som testanvändare
  2. Gå till Dashboard
  3. Klicka "Ta foto" på ett jobb
  4. Tillåt kameraåtkomst när webbläsaren frågar
  5. Ta en testbild!

Alternativ: Localtunnel (ingen varning)

Om ngrok fortsätter ha problem, använd localtunnel istället:

npm install -g localtunnel && lt --port 3000
